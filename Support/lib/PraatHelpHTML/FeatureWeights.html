FeatureWeights

One of the types of objects in Praat.

A FeatureWeights object is a d-dimensional vector containing weight values used to transform a d-dimensional space. Feature weighting can be used to improve the classifcation accuracy of KNN classifiers. It can also be used to generate a Dissimilarity matrix from a Pattern object. Dissimilarity matrices in conjunction with MDS-analysis can aid the visualization of high-dimensional data.
