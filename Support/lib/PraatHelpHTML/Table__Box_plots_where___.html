Table: Box plots where...

A command to draw box plots from the data in a column of the selected Table object.

Example

To draw separate box plots for the male, female and children F0 for the Peterson & Barney (1952) data:

Create formant table (Peterson & Barney 1952)
 Box plots where: "F0", "Type", 70, 400, "1"
 Text left: "yes", "F0 (Hz)"
