VowelEditor: Show vowel marks from Table file...

A command in the VowelEditor that lets you set your own vowel marks.

Layout of the Table

The Table needs at least three mandatory columns labeled "Vowel", "F1" and "F2" and one optional column labeled "Size". The Vowel column contains the vowel marker labels, the F1 and F2 columns have the first and second formant frequencies in Hertz. The optional Size column contains the font size of the vowel markers.
