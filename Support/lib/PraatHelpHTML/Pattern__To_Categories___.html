Pattern: To Categories...

Split the given Pattern into a fixed number of clusters using the %%k%-means clustering algorithm. A Categories object containing numbered categories corresponding to the generated clusters will be created.

Settings

k clusters
The number of clusters to be generated.
Cluster size ratio constraint
The minimum allowed ratio between the smallest and the biggest cluster.
Maximum number of reseeds
The maximum allowed number of reseeds used to enforce the cluster size ratio constraint.
